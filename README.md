# Generador de Contraseñas Seguras

Un pequeño programa en Python que te genera una contraseña segura en segundos. Sin instalaciones, sin configuraciones — solo se debe correr el script y listo.

## ¿Qué hace?

Le dices cuántos caracteres quieres (mínimo 8). Mezcla letras, números y símbolos de forma completamente aleatoria, garantizando que la contraseña siempre tenga al menos uno de cada tipo.

## Requisitos

- Python 3.6 o superior
- No necesita instalar nada extra — usa solo librerías que vienen con Python

## Posibles Mejoras

Desarrollar un apartado visual

## Limitantes

Limitado a generar una contraseña a la vez.

## Puntos a favor

No guardado de hostorial información limpia sin recomendaciones de antiguos usuarios.

## Cómo usarlo

```bash
git clone https://github.com/cristosgomez/Autonomo-2.git
cd Autonomo-2
python generador.py
```

El programa te irá guiando paso a paso desde la terminal.
